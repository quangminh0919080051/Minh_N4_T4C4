# 🌐 My Awesome Web Application

![Project Banner](https://your-image-url.com/banner.png)

## 📖 Giới thiệu

**My Awesome Web Application** là một trang web bán hàng với các tính năng như quản lý sản phẩm, đặt hàng, theo dõi đơn hàng và phản hồi khách hàng. Dự án này được phát triển bằng **React** và **Redux**, với thiết kế giao diện người dùng được xây dựng bằng **Ant Design**.

## ✨ Tính Năng
- **🔍 Tìm kiếm sản phẩm**: Người dùng có thể tìm kiếm, lọc và xem chi tiết các sản phẩm.
- **🛒 Giỏ hàng và Thanh toán**: Thêm sản phẩm vào giỏ hàng và thanh toán trực tuyến.
- **📦 Theo dõi đơn hàng**: Kiểm tra tình trạng và lịch sử đơn hàng.
- **📱 Thiết kế đáp ứng**: Giao diện thân thiện với mọi thiết bị, từ máy tính đến điện thoại di động.

## 🛠️ Công nghệ sử dụng

- **Frontend**: React, Redux, React Router
- **Backend**: Node.js, Express
- **Cơ sở dữ liệu**: MongoDB
- **Thiết kế UI**: Ant Design, CSS Module
- **Triển khai**: Heroku, Netlify
